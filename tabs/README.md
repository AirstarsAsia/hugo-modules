# Tab Shortcode

## Installation for Gethugothemes themes

Add the following code to your `asstes/scss/style.scss` file.

```scss
@import 'tabs';
```

<hr>

## Shortcode Implementation

```md
{{< tabs "demo-tab" >}}

{{< tab "first" >}}
First Tab
{{< /tab >}}

{{< tab "second" >}}
Second Tab
{{< /tab >}}

{{< tab "third" >}}
Third Tab
{{< /tab >}}

{{< /tabs >}}
```
